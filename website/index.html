Architecture

Small, secure, and easy to maintain.

01

Private S3 Storage

The website files are stored in an encrypted S3 bucket with public access blocked.

02

CloudFront Delivery

CloudFront serves the site globally over HTTPS and accesses S3 through OAC.

03

OIDC Deployment

GitHub Actions assumes a scoped IAM role without storing AWS access keys.

Infrastructure as Code

Everything is repeatable.

Terraform creates the S3 bucket, CloudFront distribution, bucket policy, GitHub OIDC provider, and deployment IAM role. Pushes to the main branch publish the contents of the website directory automatically.
